# Employee Tracker

![License: MIT](https://img.shields.io/badge/license-MIT-blue)

## Description:

A node command-line application/Content Management System used to manage a company's employee database, using Node.js, Inquirer, and MySQL.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [License](#license)
- [Questions](#questions)

## Installation:

If you already have node installed the only step for installing the project is running 'npm install' from the root directory.

## Usage:

To start the project you need to run 'node index' from the root directory. Checkout the video to see a full demonstration.
